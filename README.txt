Rodando o projeto:

Usei o servidor Apache e o banco de dados MySQl usando o XAMPP control panel

A pasta desse projeto ficou dentro da pasta htdocs para o servidor poder rodar -> C:\xampp\htdocs\projeto_xapp

Após iniciar o servidor e o banco de dados, você precisa rodar os comandos que estão em projeto_xapp\database\database.sql para fazer a criação do BD

Após isso você precisará informar suas credenciais para conexão com o DB em projeto_xapp\config\connection.php
$host = '';
$dbname = '';
$user = '';
$pass = '';

Após criar as tabelas no BD e conectar com o banco de dados, você pode rodar a aplicação na sua máquina em http://localhost/projeto_xapp/ e fazer os testes desejados.
